WHAT IS VOLUME BY TIME?

Volume by Time analyzes how volume behaves at specific times throughout
the day by:

✓ Aggregating volume data by exact timestamp
✓ Calculating historical average/median volume at each time
✓ Comparing current volume to historical norms
✓ Tracking directional volume (buy vs sell)
✓ Identifying patterns and anomalies
✓ Detecting institutional activity windows

Result: Traders know when volume is NORMAL vs UNUSUAL at any time
of day, enabling better timing and trade confirmation.

QUICK START (5 MINUTES)

from luxalgo_volume_by_time import VolumeByTimeAnalyzer, AggregationMethod
from luxalgo_volume_by_time_advanced import (
    VolumeSignalDetector, InstitutionalActivityDetector
)
import pandas as pd

# Load hourly OHLCV data
df = pd.read_csv('spy_1h.csv', index_col='timestamp', parse_dates=True)

# Create analyzer (0 days = all history, or specify lookback)
vbt = VolumeByTimeAnalyzer(
    lookback_days=0,  # Use all history
    method=AggregationMethod.AVERAGE  # or MEDIAN
)

# Analyze
df_result, analyses = vbt.analyze(df)

print(f"✓ Bars analyzed: {len(analyses)}")

# Check current bar
if analyses:
    latest = analyses[-1]
    print(f"Time: {latest.time_slot}")
    print(f"Volume: {latest.current_volume:,.0f}")
    print(f"Historical Avg: {latest.historical_average:,.0f}")
    print(f"Above Average: {latest.is_above_average}")
    print(f"Anomaly: {latest.volume_anomaly*100:+.1f}%")

# Detect patterns
accum = VolumeSignalDetector.detect_accumulation(analyses)
dist = VolumeSignalDetector.detect_distribution(analyses)

if accum:
    print(f"\n✓ Accumulation detected: {accum.confidence:.0f}%")
if dist:
    print(f"\n✓ Distribution detected: {dist.confidence:.0f}%")

# Institutional activity
moc = InstitutionalActivityDetector.identify_moc_activity(analyses)
if moc:
    print(f"\n✓ MOC Activity: {moc['bias'].upper()}")
    print(f"  Volume: {moc['total_volume']:,.0f}")

HOW IT WORKS (5 STEPS)

Step 1: EXTRACT TIME SLOTS
  Get exact time from each bar (15:30, 15:31, etc.)
  Group bars by time of day across all sessions

Step 2: AGGREGATE VOLUMES
  Collect all volumes for each time slot
  Calculate average or median
  Store up and down volume separately

Step 3: TRACK DIRECTIONAL VOLUME
  Determine if bar was up (close > open) or down
  Estimate proportion of up vs down volume
  Build historical profile of buy/sell at each time

Step 4: CALCULATE ANOMALIES
  Compare current bar volume to historical average
  Calculate percentage deviation (anomaly)
  Flag if above/below normal

Step 5: DETECT PATTERNS
  Accumulation: High buy volume
  Distribution: High sell volume
  Breakout: Volume spike (statistical outlier)
  Exhaustion: Declining volume

  15:00 PM - Hollow: 300k, Solid: 400k (Spike! ✓✓)

═══════════════════════════════════════════════════════════════

AGGREGATION METHODS

AVERAGE (Default)
  ✓ Smooth and responsive
  ✗ Influenced by outliers
  Use when: Most market conditions
  Example: 250k average over 50 sessions

MEDIAN
  ✓ Robust against outliers
  ✓ True typical behavior
  ✗ Slower to react to changes
  Use when: Volatile markets with spikes
  Example: 240k median (filters 50k spike days)

Switch methods:
  vbt.set_method(AggregationMethod.MEDIAN)

BI-DIRECTIONAL VOLUME

Track buy and sell volume separately:

CURRENT_UP_VOLUME
  Volume on bars where close > open
  Aggressive buying pressure
  
CURRENT_DOWN_VOLUME
  Volume on bars where close < open
  Aggressive selling pressure

BUY/SELL RATIO
  total_up / total_down
  >1.0 = More buying
  <1.0 = More selling

AGGRESSIVE BUYING
  Up volume > 65% of total = Institutional buying
  
AGGRESSIVE SELLING
  Down volume > 65% of total = Institutional selling

VOLUME PATTERNS (4 TYPES)

ACCUMULATION
  Setup:
  - Buy volume > 65% of total
  - Sustained over 5-10 bars
  - Confidence: Increases with strength
  
  Interpretation:
  ✓ Institutions accumulating position
  ✓ Bullish bias building
  ✓ Consolidation phase ending
  
  Action:
  - Watch for breakout confirmation
  - Enter on close above resistance
  - Size: Full position (high confidence)

DISTRIBUTION
  Setup:
  - Sell volume > 65% of total
  - Sustained over 5-10 bars
  - Confidence: Increases with strength
  
  Interpretation:
  ✓ Institutions distributing position
  ✓ Bearish bias building
  ✓ Top formation in progress
  
  Action:
  - Wait for breakdown confirmation
  - Short on close below support
  - Size: Full position (high confidence)

BREAKOUT (Volume Spike)
  Setup:
  - Volume >1.5σ from average
  - Unusual activity at specific time
  - Statistical outlier detected
  
  Interpretation:
  ✓ Unexpected volume surge
  ✓ Breaking institutional balance
  ✓ Major move about to start
  
  Action:
  - Fade initial move (30% position)
  - Add on continuation (70% position)
  - Tight stop initially

EXHAUSTION (Declining Volume)
  Setup:
  - Volume declining over 5 bars
  - Below 70% of average
  - Momentum fading into move
  
  Interpretation:
  ✓ Move running out of fuel
  ✓ Late arrivals entering
  ✓ Reversal setup forming
  
  Action:
  - Reduce position size
  - Prepare to reverse
  - Move stops to breakeven

TROUBLESHOOTING

Q: "No patterns detected"
A: 1. Increase lookback period (get more data)
   2. Lower pattern confidence threshold
   3. Use different time aggregation (15-min instead of 1H)
   4. Check market is trending (patterns rare in low-vol)

Q: Too many false signals?
A: 1. Increase confidence threshold (>75%)
   2. Combine with price action confirmation
   3. Use median instead of average
   4. Add volume spike filter

Q: "Historical average is NaN"
A: 1. Not enough samples at that time
   2. Increase lookback period
   3. Use coarser time buckets (15-min vs 1-min)
   4. Ensure data spans multiple days

Q: Accumulation/Distribution not working?
A: 1. Verify directional volume is calculated
   2. Check buy/sell ratio threshold (65% default)
   3. Increase lookback window (5→10 bars)
   4. Use on trending timeframes, not ranging
