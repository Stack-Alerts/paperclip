WHAT IS SWING BREAKOUT SEQUENCE?

Swing Breakout Sequence identifies the classic "two failed breakouts then
successful third" pattern. It detects when price attempts to break through
a swing zone twice but fails, then sets up for a successful third attempt.

✓ 5-Point Structure - Complete breakout sequence from setup to entry
✓ Failed Breakout Detection - Identifies both pullbacks and reversals
✓ Liquidity Traps - Deeper pullbacks that trap stop losses
✓ Reversal Patterns - Double tops/bottoms at Point 5
✓ Two Failed Attempts - Required before third (successful) breakout
✓ High Probability - 60-75% win rate on third breakout
✓ Multi-Timeframe - Works 1H+ (best 4H-daily)

Result: Trade the third breakout when first two have failed.

QUICK START (5 MINUTES)

from luxalgo_swing_breakout_sequence import SwingBreakoutSequenceDetector
from luxalgo_swing_breakout_sequence_advanced import (
    SequenceTradeAnalyzer, SequenceReporter
)
import pandas as pd

# Load OHLCV data
df = pd.read_csv('eurusd_4h.csv', index_col='date', parse_dates=True)

# Create detector
detector = SwingBreakoutSequenceDetector(
    swing_length=5,
    internal_length=3,
    require_point_4=True,
    require_point_5=True,
    point_4_beyond_point_2=True
)

# Detect sequences
df_result, results = detector.analyze(df)

print(f"✓ Total Sequences: {results['total_sequences']}")
print(f"  Bullish: {results['bullish_sequences']}")
print(f"  Bearish: {results['bearish_sequences']}")
print(f"  Complete (with Point 5): {results['complete_sequences']}")

# Analyze trades
trades = SequenceTradeAnalyzer.generate_trades(
    results['sequences'], df_result,
    entry_on_point_5=True
)

stats = SequenceTradeAnalyzer.calculate_statistics(trades)
print(f"\n✓ Win Rate: {stats['win_rate']:.1f}%")
print(f"  With Point 5: {stats['with_point_5_win_rate']:.1f}%")
print(f"  Without Point 5: {stats['without_point_5_win_rate']:.1f}%")

# Report
reporter = SequenceReporter()
print(reporter.generate_trade_report(trades))

THE 5-POINT SEQUENCE STRUCTURE

POINT 1: INITIAL BREAKOUT ATTEMPT
  What: Price breaks outside swing zone for the first time
  Bullish: Breaks above swing high
  Bearish: Breaks below swing low
  Duration: Usually 1-3 bars to breakout
  Psychology: Traders buying/selling the breakout
  Interpretation: First wave of momentum

POINT 2: PULLBACK INTO ZONE
  What: Price returns back into swing zone
  Bullish: Pullback to swing zone after break
  Bearish: Pullback to swing zone after break
  Duration: Usually 2-5 bars
  Psychology: Initial breakout traders getting shaken out
  Interpretation: Sellers/buyers taking profits
  KEY: This marks the first trap/failure

POINT 3: SECOND BREAKOUT ATTEMPT
  What: Price attempts to break again (after Point 2 pullback)
  Bullish: Tries to break above swing again
  Bearish: Tries to break below swing again
  Duration: Usually 1-3 bars
  Psychology: Retail traders seeing double breakout
  Interpretation: "This time it will work" mentality
  KEY: Setup for second trap

POINT 4: DEEPER PULLBACK (LIQUIDITY TRAP)
  What: Price returns into zone DEEPER than Point 2
  Bullish: Goes lower than Point 2 low
  Bearish: Goes higher than Point 2 high
  Duration: Usually 2-5 bars
  Psychology: Traders hold stops at Point 2 level (get hit)
  Interpretation: Liquidity sweep/stop hunt
  KEY: This is the liquidity trap zone
  Optional: Can disable if too few sequences detected

POINT 5: REVERSAL PATTERN (Signal!)
  What: Double top/bottom forms at Point 4 level
  Bullish: Double bottom at Point 4 lows
  Bearish: Double top at Point 4 highs
  Duration: Usually 1-2 bars
  Psychology: Market rejection of Point 4 level
  Interpretation: Setup now ready for 3rd breakout
  KEY: ENTRY SIGNAL - Reversal confirmed
  Optional: Can require or disable

CONFIGURATION & SETTINGS

SWING LENGTH (Default: 5)
  Definition: Bars needed to confirm swing high/low
  Range: 3-21 bars
  Higher (13-21):
    ✓ Larger swings, fewer false signals
    ✗ Fewer sequences detected
    Use: Position trading, higher probability
  
  Lower (3-5):
    ✓ More swings, more sequences
    ✗ More false signals
    Use: Day trading, scalping

INTERNAL LENGTH (Default: 3)
  Definition: Bars for pattern recognition at Point 5
  Range: 1-5 bars
  Higher (5):
    ✓ Stricter pattern detection
    ✗ Fewer Point 5 patterns
  
  Lower (1-2):
    ✓ More patterns detected
    ✗ Less strict

REQUIRE POINT 4 (Default: ON)
  Definition: Require deeper pullback past Point 2
  Impact: Only detect liquidity trap sequences
  Effect: Fewer sequences, higher quality
  Recommendation: ON for best results

REQUIRE POINT 5 (Default: ON)
  Definition: Require reversal pattern at Point 4
  Impact: Only trade confirmed reversals
  Effect: Better entry confirmation
  Recommendation: ON for entries

POINT 4 BEYOND POINT 2 (Default: ON)
  Definition: Point 4 must exceed Point 2
  Impact: Filters true liquidity traps
  Effect: Higher win rate
  Recommendation: ON

Q: "Too few sequences detected"
A: 1. Lower swing length (try 3-5)
   2. Disable Point 4 requirement temporarily
   3. Disable Point 5 requirement temporarily
   4. Check data quality (verify OHLC)

Q: "Win rate much lower than expected"
A: 1. Enable Point 4 requirement (filters noise)
   2. Enable Point 5 requirement (confirms reversal)
   3. Only trade Point 5 entries (higher win rate)
