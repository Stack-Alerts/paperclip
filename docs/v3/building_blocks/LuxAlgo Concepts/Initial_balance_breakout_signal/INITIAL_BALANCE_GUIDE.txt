## 🎯 WHAT IS INITIAL BALANCE?

The **Initial Balance (IB)** is the high and low price range during the **first hour** of a trading session.

### Why It Matters:
- ✅ Institutional traders accumulate/distribute during first hour
- ✅ Sets tone for rest of trading day
- ✅ Breakouts above/below IB = momentum continuation
- ✅ Rejections at IB = mean reversion opportunities
- ✅ Used for Open Range Breakout (ORB) strategies

### Example:
```
Regular market hours: 9:30 AM - 4:00 PM
Initial Balance: 9:30 AM - 10:30 AM (first 60 minutes)
  
If IB High = $100.50 and IB Low = $100.00:
  - Price above $100.50 → Bullish breakout signal
  - Price below $100.00 → Bearish breakout signal
```

## 📈 TRADING STRATEGY PATTERNS

### Pattern 1: ORB (Open Range Breakout)
```python
# Configure for 5-min open range
detector = InitialBalanceDetector(
    session_type=SessionType.ORB_5MIN,
    session_duration_minutes=5,
)

# Find breakout of first 5 minutes
ib = detector.detect_initial_balance(df)
signals = detector.detect_breakouts(df, ib)

# Trade the breakout
for signal in signals:
    if signal.direction == BreakoutDirection.BULLISH:
        entry = signal.price
        stop = ib.low
        target = detector.calculate_extensions(ib).top_50
        print(f"LONG: Entry {entry:.2f}, Stop {stop:.2f}, Target {target:.2f}")
```

### Pattern 2: IB Retest Bounce
```python
# Watch for price to come back and test IB extremes
# Then bounce in opposite direction

fibs = detector.calculate_fibonacci_levels(ib)
retracement_50 = fibs.level_50_0  # Midpoint

# If price approaches IB midpoint, expect bounce
if abs(df['close'].iloc[-1] - retracement_50) < (ib.range_size * 0.02):
    print("Price at IB midpoint - expect bounce to IB extreme")
```

### Pattern 3: Fibonacci Extension Profits
```python
exts = detector.calculate_extensions(ib)

# For breakout trades, use extensions as profit targets
if signal.direction == BreakoutDirection.BULLISH:
    # Take partial profits at each extension
    first_profit = exts.top_25
    second_profit = exts.top_50
    extended_profit = exts.top_100
```

### Pattern 4: Multi-Session Confluence
```python
past_ibs = detector.get_past_ibs(df)

# Check if today's IB aligns with previous sessions
for past_ib in past_ibs:
    if abs(ib.high - past_ib.high) < (ib.range_size * 0.05):
        print("Today's IB high matches previous session - strong resistance")
```

## 🎓 TRADING CONCEPTS

### Initial Balance High/Low
- **IB High**: Resistance for bearish traders, breakout level for bullish
- **IB Low**: Support for bullish traders, breakout level for bearish

### Fibonacci Ratios
- **23.6%**: Shallow retracement, trend continuation likely
- **38.2%**: Common support/resistance level
- **50%**: Midpoint, equilibrium, mechanical support
- **61.8%**: Deep retracement, reversal likely
- **78.6%**: Very deep, often confirms reversal

### Extensions
- **25%**: Conservative profit target
- **50%**: Standard breakout target
- **100%**: Aggressive extended target

### Forecast Methods
- **vs Previous Open**: Compare today's IB to where yesterday opened
- **Filter by Weekday**: Compare to average Tuesday/Wednesday/etc

---

## 🚨 COMMON MISTAKES

❌ **Too sensitive**: Detecting noise as breakouts
✅ **Solution**: Increase volume multiplier or confirmation bars

❌ **Missing entries**: Not catching early breakouts
✅ **Solution**: Decrease confirmation_bars from 2 to 1

❌ **Wrong session time**: Analyzing wrong market hours
✅ **Solution**: Verify session_start_hour matches your market

❌ **Ignoring volume**: Acting on weak volume signals
✅ **Solution**: Always use volume_multiplier > 1.5
