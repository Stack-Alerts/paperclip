## Operational Flow

```
Every 5 minutes (GitHub Actions scheduled):
├─ Fetch all done issues from Paperclip API
├─ Filter to fix/bug issues only
├─ For each issue:
│  ├─ Check comments for Impact Gate result
│  ├─ If already gated → skip
│  └─ If ungated → run full Impact Gate
├─ Generate JSON summary
├─ Upload artifact (30-day retention)
├─ Write step summary
└─ Sleep 300 seconds
```
